This is the software I use on my machine for work and other things.

## Operating System
As my OS of choice I use Fedora, as I find it less frustrating than Debian and Arch. Arch is unstable and prone to breakage, meanwhile I couldn't even install lazy.nvim on Debain due to the outdated neovim version.

## DE/WM
I personally prefer KDE to other Desktop Environments and Window Managers, because it is modern and has all the necessary features with a large ecosystem of apps. I tried using tiling WMs countless times (i3, sway, dwm, awesome, hyprland), but for me, they're more of a pain in the ass rather than an actual productivity gain. I don't really see the difference between using Win+1/2 or Alt+Tab and I don't want to edit my dotfiles in order to change the display layout.

## IDE/Text Editor
I use Neovim as my main development tool. Everything is done inside a tmux session, which I don't use any special config for. My shell is zsh with the [Oh My Zsh!](https://ohmyz.sh/) config.
